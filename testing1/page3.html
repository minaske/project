Cara melakukan pemesanan Foto

  1. Pilih menu "Cetak"
  2. Kirim file berupa jpeg/png
  3. Jawab beberapa pertanyaan yang tersedia dengan benar
  4. Isi alamat dan no telpon anda
  5. Berikan desrpisi tambahan (jika ada)
  6. Lalu kirim file
